Документация CrunchAPI

Модуль CrunchAPI предназначен для взаимодействия с веб-сайтом CodeCrunch.

Обзор

Кратко: CrunchAPI это библиотека разработанная создателями CodeCrunch для удобного взаимодействия с тестами и информацией тестов. Написана без использования не штатных библиотек.

Установка

pip install CrunchAPI

Одна команда для установки огромного пакета возможностей CodeCrunch!

Использование

fetch_data(url)

Метод для отправки HTTP GET-запросов. Метод используется для работы двух других, прошу вас его не трогать!

getTest(id)

Метод для получения информации о тесте по его идентификатору. Как аргумент принимает ID теста, возвращает лист всех названий, id, ответов на тест . 

from CrunchAPI import Test

test = Test()
print(test.getTest(11))

getAllId()

Метод для получения списка всех идентификаторов тестов. Возвращает массив чисел - всех идентификаторов тестов ! 


        from CrunchAPI import Test

        test = Test()

        print(test.getAllId())

addTest()

Метод для добавления теста на CodeCrunch. Принимает в виде аргумента словарь состоящий из следующих ключей: testName, testDescription, questions, options[], selectedOption 


from CrunchAPI import Test

test = Test()

info = {'testName': 'name of your test',
         'testDescription': 'description of your test',
           'question': 'qst',
             'options': ['pick 1', 'pick 2', 'pick 3', 'pick 4', 'pick 5'],
               'selectedOption': 3}
print(test.addTest(info))
